Lesson 1 The Basics of the Web and HTML

HTML

HTML stands for HyperText Markup Language and is used to process communication across the Internet. HTTP is the protocol used to communicate between my computer and servers.The world wide web is collection of HTML Documents. Major components of the Internet include the following:

  1. My computer/browser
  2. The Internet
  3. Servers
  4. HTTP

Inline vs Block Elements

Elements are either inline or block. Block elements form an "invisible box" around the content inside of them.

What a Web Page is

A web page is a text document written in a language called HTML. Web browsers read these documents, and then interpret and display them.

How Coding Works

Coding happens when programmers write text in a language that a computer can understand. The computer can then follow the instructions the programmer wrote. For example, the computer might do this by making text like this:
"I'm learning to code!"
look like this:
"I'm learning to code!"

Computers are Stupid

Programmers need to write exactly the way a computer understands (also known as writing with correct "syntax").

Programmers Can't Remember Everything

There are too many details to keep everything in your head. And that's okay. If you forget how to make text italic in HTML, you can always just look it up.

Basic HTML Vocabulary

You will be using HTML in the next few lessons, so it will be helpful if you're comfortable with the jargon.

  1. Tag: An HTML tag is always contained within angled brackets. Most tags have an opening tag. Some tags (called "void" tags) do not require a closing tag.
  2. Element: An HTML element refers to everything within a set of opening and closing tags.
  3. Attribute: This is a property of an HTML element.

Lesson 2 Creating a Structured Document with HTML

Developer Tools (in the Browser)

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them.

The "tree-like structure" of HTML

The "tree-like structure" comes from the fact that HTML elements can have other elements inside of them. You can draw this relationship like a family tree. My mother had multiple children. So did her mother, and so on...In a browser, this structure shows up as a series of nested boxes. There are boxes inside of boxes inside of boxes, and so on...

The relationship between indented HTML and boxes

When you read an HTML document as text, you see a wave of changing indentations going up and down the left side of the document. The more indented an element is, the more deeply nested it's corresponding "box" is.

Text Editors (for programming)

When writing code, programmers use special text editors (like Sublime Text for example). These editors make the programmer's life easier. For example, some text editors will automatically generate a closing HTML tag when you write an opening tag.

Lesson 3 Adding CSS Style to HTML Structure

Style Tag Example

You can use the style tag to apply style as follows:

This text should be blue

CSS

CSS = Cascading Style Sheets

CSS controls the style of elements in HTML

CSS allows programmers to eliminate repetitive code decreasing the chances for errors. When possible repeated code should be eliminated to as mentioned above decrease the opportunity for errors to occur.

Methods for including CSS Styling in Web Pages

Include CSS in the heading of HTML - This Method is good for smaller projects

Link HTML to a separate CSS - This requires additional steps but is more organized.

Write your style inline with your HTML - Typically a bad idea creating a lot of repeated code.
